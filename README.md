A simple web application that lets you view and listen to a connected capture card directly in your browser.

## Features

- Live video preview
- Audio playback
- Works entirely in the browser
- No installation required
- Lightweight single-page application

## Requirements

- A modern web browser
- A supported capture card connected to your computer
- Permission to access media devices

## Usage

1. Open `index.html` in your browser.
2. Allow device access when prompted.
3. Select your capture card.
4. View and listen to the capture card feed.

## Files

```text
.

```

## Notes

This application uses the browser's MediaDevices API to access video and audio streams from supported capture devices. All processing is performed locally in your browser.
